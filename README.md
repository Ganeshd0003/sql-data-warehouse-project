# Data Warehouse Project

## Project Overview
Build a modern data warehouse using SQL Server for sales analytics.

## Objectives
- Extract raw data
- Clean and transform
- Build Star Schema
- Load warehouse
- Create analytical views
- Generate business insights

## Tech Stack
- SQL Server
- SSMS
- Git
- Draw.io (ER Diagram)

## Architecture
CSV Files
    │
    ▼
Bronze Layer
(Raw Data)
    │
    ▼
Silver Layer
(Clean Data)
    │
    ▼
Gold Layer
(Dimension + Fact Tables)
    │
    ▼
Analytics
